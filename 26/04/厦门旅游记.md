即使在厦门无比清晰地记住每个细节，才投入工作一小会儿，这些细节就演变成了模糊的过去。我刚才发现，厦门到底玩了几天我都记不清楚了，我来理一理。

## Day 1 周六
早上赶浦东机场

=> 厦门，打车到集美区：

到的时候差不多两点。同样的价格和差不多的环境，我相比民宿可能更偏向于酒店，因为那里人味儿少。不过最近住的几个民宿特别干净拉高了印象分，没有人味儿的同时还给我一种我就是当地居民的感觉，如果阳台种点花更是如此，这是民宿最大的优势。

老板给我们沿路介绍集美大社，感觉这里就像深圳湾旁边的弄堂区，独立经营的书屋民宿和照相馆有各种贴纸和奇怪动静，很有闲钱搞艺术搞创作的感觉。东西也不贵。才逛了大社半小时，我就说：这次来厦门彻底值了，现在回上海我都觉得没亏，在杭州只有河坊街那种臭豆腐街。

到出大社的时候我还觉得这里的商业极有个性；然而后来回顾才发现，其实也就比河坊街好了一点，很多东西实际上也是量产的没有什么特色。

集美的交通让我对这个区的评价从5星降到2星，这是我见过史上最差的通行环境，没有非机动车道，人行道就半米多宽，所有电动车都在跟汽车抢道，车道又窄，喇叭动静贼大，红绿灯一共没几个，上下坡坡度大地面还光滑（还下雨了），飙车的一大堆，所有车道都有海量逆行的

=> 千与千寻海滩，其实就是看桥柱，不幸的是当天阴天。并且看起来只是一个大湖湾，我想看海来着。后来去了没啥记忆点的集美大学等地方。以及楚雨寻的家，也不知道现在谁在住，我看一次笑一次。

=> 晚上先睡了一会儿，十点多根据老板推荐出去吃了朱富贵海鲜，开始觉得便宜又好吃，但这配料太单薄，海鲜也不怎么新鲜，很多东西就是，第一口你能打满分，吃到一半就很腻了，不好吃。最后居然吃了三百多，我跟 lmz 点的东西太多重复了，重头戏皮皮虾非常小只，而且因为自助火锅，本质就是所有海鲜用小锅清水煮，没有爆炒调味料是真难受，我一开始用捞汁吃法，太腻了到底是谁在喜欢捞汁，后来不得不上醋防止太腻

没想到下午到还能玩这么多东西。

## Day 2 周日

=> 早上七八点起床去赶鼓浪屿船票

=> 鼓浪屿的可乐冰沙只要5块钱，天气又闷又热我俩吃个爽。我觉得这里小镇风格不错，新旧适中，四通八达，路线有点像逛我老家村子，建筑有点像玩箱亭游戏

=> 先补觉，下午出发去海滩。这里看海还是像海湾，没有一望无际的感觉，我们穿过了几个沙滩，在上面写字儿，沙滩尽头是菽庄花园，很无聊，还把我拍立得摔坏了

拍立得在摔坏之前，拍了很多好看的照片，这质感你说这是 2000 年我都信

=> 逛鼓浪屿那个商业街，千篇一律全是连锁小店，一点意思都没有。颜值比杭州垃圾街高一点，看到那种统一印制的广告牌，价格图片都一样，我一点胃口没有，差点吃了肯德基，最后还是离开肯德基去吃了一个小生蚝汤（不是生蚝，叫啥来着）

=> 最美转角之类的地标给我看笑了。但是正是因为无聊而又被游客哄抢，这种奇异的 vibe 在安静的海边还是有点让人欢乐

=> 晚上为了给自己和 lmz 采购物品，撑伞四处求索。她后来在写作业，我看 KPL

## Day 3 周一

=> 上午离开鼓浪屿，把东西放到遥远的酒店，首先去了附近的黄厝海滩。OK，这是我见过最 OK 的海滩好吧，除了远处小海岛，基本上是99%的天际线，即使是多云到阴，也比较好看。

=> 沿着沙滩捞贝壳，以为能给龟家装饰装饰。我说，自己捞的贝壳那肯定有加分

=> 后来下雨了，去厦大，狂下雨，全身湿的逛厦大。造的挺好看的还有地下商业街，不过也比较小

=> 植物园。这什么玩意儿，裸子植物区就是一个没咋维护过的小山？种了一些铁树？多肉植物区就是仙人掌？想必晴天的时候还算美丽，但是咱们这时间挑的太阴间

我说：照这么说一个仙人掌就是南美植物区 + 非洲植物区 + 热带植物区 + 多肉植物展览馆 + 仙人掌展览馆

=> 晚上去中山街吃的东西被坑了，出租车司机您就别推荐了，我很早就觉得当地人咬定某某饭馆好吃实惠是不靠谱的，因为那是你当地人口味或者你自己家里爱吃，你也不了解我是从哪儿来的你也不知道我口味，咋那么有自信的，还说别的都是坑吃了也是犯傻。haha

=> 八市：千篇一律就那样儿，不过 lmz 爱吃怪味儿冰粉

=> 就是晚上为了赶海，没有去灯光长廊，没有去各种地方，先去了臭水沟沙滩，辛辛苦苦抓了一只1cm小螃蟹护了一路，随着时间流逝，不想要明早赶飞机困死，再加上到上海就是工作，玩耍的心态也附带时间的压力

=> 重头戏来了，为了继续赶海我们根据小红书去椰风寨沙滩，然而所到地沙滩坡度较大，没有平坦的沼地栖息海洋生物。我们就这样穿着拖鞋穿越无人的海滩，高昂浪声不断，你看向大海的时候，除了灯塔的光点和沙滩的警示灯什么都没有。你很少能处于一个不仅看不见东西，也确实不存在任何东西的空间。你如果往前走，就是太平洋无尽的水
