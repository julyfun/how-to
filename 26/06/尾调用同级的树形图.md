```
展示函数调用链. 使用尾调用同级的树形图，即每行一个函数或一个重要循环，不重要可省略，并且一个函数的子函数中的最后一个与父函数必须处于同一缩进，其他缩进 + 1 e.g.
f()
├── f 的第一个子函数
│   如果 f 只有一个要展示的子函数就这样画
├── ...
f 的最后一个子函数 g()
├── g 的第一个子函数
g 的最后一个子函数
```

```zig
std.debug.print()
├── std.debug.lockStderr()
│   ├── Io.lockStderr()
│   │   ├── Threaded.lockStderr()
│   │   Threaded.initLockedStderr()
│   Io.LockedStderr.clear()
├── Io.Writer.print()
│   ├── Io.Writer.writeAll()
│   │   Io.Writer.write()
│   Io.Writer.printValue()
│   Io.Writer.writeAll()
std.debug.unlockStderr()
├── Io.unlockStderr()
Threaded.unlockStderr()
├── Io.Writer.flush()
│   Writer.defaultFlush()
│   File.Writer.drain()
│   File.Writer.drainStreaming()
│   File.writeStreaming()
│   Io.operate()
│   Threaded.operate()
│   Threaded.fileWriteStreaming()
│   posix.system.writev() # 这里调用 macos 平台函数
mutexUnlock()
```

---

不使用上述方法：

```
展示函数调用链. 使用树形图，即每行一个函数或一个重要循环，不重要可省略 e.g.
f()
├── std.debug.lockStderr()
├── Io.Writer.print()
│   ├── Io.Writer.writeAll()
│   │   └── Io.Writer.write()
│   └── Io.Writer.printValue()
│       └── Io.Writer.writeAll()
└── std.debug.unlockStderr()
```

```zig
std.debug.print()
├── std.debug.lockStderr()
│   ├── Io.lockStderr()
│   │   ├── Threaded.lockStderr()
│   │   └── Threaded.initLockedStderr()
│   └── Io.LockedStderr.clear()
├── Io.Writer.print()
│   ├── Io.Writer.writeAll()
│   │   └── Io.Writer.write()
│   └── Io.Writer.printValue()
│       └── Io.Writer.writeAll()
└── std.debug.unlockStderr()
    ├── Io.unlockStderr()
    └── Threaded.unlockStderr()
        ├── Io.Writer.flush()
        │   └── Writer.defaultFlush()
        │       └── File.Writer.drain()
        │           └── File.Writer.drainStreaming()
        │               └── File.writeStreaming()
        │                   └── Io.operate()
        │                       └── Threaded.operate()
        │                           └── Threaded.fileWriteStreaming()
        │                               └── posix.system.writev() # 这里调用 macos 平台函数
        └── mutexUnlock()
```
