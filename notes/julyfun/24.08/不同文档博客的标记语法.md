# 不同文档博客的标记语法

## github 标准

```
> [!NOTE]
>
> Markdown here
```

## mkdocs material

```
!!! note "Title"

    Markdown here
```

## obsidian

````
```ad-note
Markdown here
```
````

## readthedocs

```
.. note::

    Markdown here
```
