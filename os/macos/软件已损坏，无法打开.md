# 软件已损坏，无法打开

ref: https://blog.csdn.net/m0_49562857/article/details/128998691

Try something like this:

```
sudo xattr -d com.apple.quarantine "/Applications/PicGo.app"
```

> Useful for Picgo 2.3.1 on macos arm
