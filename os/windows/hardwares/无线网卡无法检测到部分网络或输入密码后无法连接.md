# 无线网卡无法检测到部分网络或输入密码后无法连接

- 发生于华硕 B650MK 主板的 win11 上

## 找不到网络？

- 购买双频网卡，别买 2.4GHz 的只能连手机热点

## Sol

- 注意插入无线网卡后可能几十秒才有反应，系统通知会让你从中安装驱动
- 尝试关闭 VPN 或关闭 VPN 的系统代理等选项
- 尝试重启
- 尝试换 USB 接口插

> 我将 USB 无线网卡从一个 USB3.0 换成另一个 USB3.0，就能连了。曾经重启也能连。
