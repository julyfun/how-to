# Switch default user

In windows terminal:

ref: https://blog.csdn.net/qq_37085158/article/details/131041223

```
ubuntu.exe config --default-user <user_name>
```

> 可能 ubuntu.exe 没有，需切换为 ubuntu2204.exe 等。
