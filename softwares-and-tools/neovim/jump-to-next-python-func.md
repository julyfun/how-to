有时候 [m 会跳到大括号。

那你就装个这个插件。

https://github.com/jeetsukumaran/vim-pythonsense

## 使用 VimPlug 安装此插件

在 init.vim 中加到 call#begin 后

```
https://github.com/jeetsukumaran/vim-pythonsense
```

接着 run:

```
:PlugInstall
```
