## autojump

命令行按照历史记录快速跳转文件夹。

https://github.com/wting/autojump

记得要 source /usr/share/autojump/autojump.fish

- How to Install

```
git clone git://github.com/wting/autojump.git
cd autojump
./install.py or ./uninstall.py
```
