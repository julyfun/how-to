### Double-DQN
离散 critic: `obs -> value[b, num_actions]` 输出所有 action 的价值.
```python
s, a, r, nxt_s, done = replaybuffer.sample()
critic_loss = mse(
    online_critic(s, a),
    [no grad] r + gamma * target_critic(nxt_s)[argmax_a online_critic(nxt_s, all a) * (1 - done)])
)
```
